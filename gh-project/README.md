# QA Performance System v2.0

## Setup Local
npm install
npm run dev

## Deploy a Vercel
1. Push repo a GitHub
2. vercel.com > New Project > Import repo
3. Click Deploy

## Deploy a GitHub Pages
1. npm run build
2. Push carpeta dist/ a rama gh-pages
3. Settings > Pages > Source: gh-pages

## Stack
React 18 + Vite 5 + Recharts. Zero backend.
